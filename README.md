# raindrop-weekly-briefing
AI-powered weekly digest / triage of your Raindrop.io bookmarks. Click Raindrop's "save for later" browser icon, and on Sunday, you'll get an email summarizing all the articles you saved. No coding required to install, but does require moving the code to your script.google.com account and then inserting your own Raindrop/Claude API keys and email address. See: https://davidzax.substack.com/p/how-to-get-a-weekly-email-digest for full instructions!
